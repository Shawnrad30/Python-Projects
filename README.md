**Python Projects (Updated Apr 2026)**

This repository contains five Python projects of increasing complexity, developed as part of the “Python Programming for Beginners” course by YouTuber and data analyst Alex Fredberg. Each project is organized into its own folder and demonstrates different concepts, ranging from basic programming to file handling and web scraping.

**Overview:**
- **Project 1:** *Unit of Measurement Converter*  
  A simple tool for converting between different units of measurement

- **Project 2:** *Calculator*  
  A basic calculator application that performs arithmetic operations

- **Project 3:** *Automatic File Sorter*  
  Organizes files into folders based on their type (CSV, images, text, etc.)

- **Project 4:** *Automated Web Scraper*  
  Collects and stores cryptocurrency data from the web

- **Project 5:** *Web Scraping & Regex*  
  Analyzes text data using web scraping and regular expressions
